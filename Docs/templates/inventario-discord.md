# Inventario de Discord

Use este arquivo para mapear servidores/canais que hoje puxam sua atencao.

## Unidade de exemplo preenchida

```markdown
## Servidor: Exemplo Game Oficial

- canal: #announcements
- objetivo real: acompanhar updates oficiais do jogo
- frequencia de consulta atual: diaria
- tipo de informacao util: patch notes, eventos, manutencao
- fonte externa equivalente: pagina Steam news ou site oficial
- substituto RSS possivel: sim
- manter no Discord: talvez, apenas se houver noticia exclusiva
- notificacoes: desligar tudo exceto mencoes importantes
- risco de distracao: alto
- acao sugerida: procurar feed oficial e testar por 1 semana.
```

## Template para preencher

```markdown
## Servidor: <nome>

- canal:
- objetivo real:
- frequencia de consulta atual:

- notificacoes: manter / reduzir / desligar

```
