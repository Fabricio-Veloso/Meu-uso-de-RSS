# Inventario de jogos

Use este arquivo como modelo. Copie uma unidade para cada jogo importante.

## Unidade de exemplo preenchida

```markdown
## Team Fortress 2

- prioridade: alta
- link Steam: https://store.steampowered.com/app/440/Team_Fortress_2/
- appid Steam: 440
- onde acompanho hoje: Steam, Discord, YouTube
- tipo de update que importa: patch notes, eventos, itens novos, grandes updates
- frequencia desejada: quando sair update
- feed candidato: https://store.steampowered.com/feeds/news/app/440/?cc=BR&l=portuguese
- precisa Discord: talvez
- risco de distracao: medio
- observacoes: testar feed Steam antes de manter servidor Discord para updates.
```

## Template para preencher

```markdown
## <nome do jogo>

- prioridade: alta / media / baixa
- link Steam:
- appid Steam:
- onde acompanho hoje: Steam / Discord / YouTube / site / outro
- tipo de update que importa: patch notes / eventos / devlog / promocoes / competitivo / outro
- frequencia desejada: imediata / diaria / semanal / tanto faz
- feed candidato:
- precisa Discord: sim / nao / talvez
- risco de distracao: alto / medio / baixo
- observacoes:
```
