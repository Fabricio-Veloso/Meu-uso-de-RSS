# Inventario de fontes

Use este arquivo para listar sites, blogs, newsletters, canais YouTube e fontes profissionais/pessoais.

## Unidade de exemplo preenchida

```markdown
## Martin Fowler

- categoria: Engenharia de software
- link: https://martinfowler.com/
- tipo: blog/site
- idioma: ingles
- motivo para acompanhar: arquitetura, design de software, refatoracao
- frequencia esperada: baixa/media
- feed RSS: https://martinfowler.com/feed.atom
- valor esperado: alto
- risco de distracao: baixo
- decisao: adicionar no leitor RSS
```

## Template para preencher

```markdown
## <nome da fonte>

- categoria: noticias / engenharia de software / AI-tech / games / pessoal / outro
- link:
- tipo: site / blog / newsletter / YouTube / Reddit / Discord / outro
- idioma: portugues / ingles / outro
- motivo para acompanhar:
- frequencia esperada: alta / media / baixa
- feed RSS:
- valor esperado: alto / medio / baixo
- risco de distracao: alto / medio / baixo
- decisao: adicionar / testar / descartar / pesquisar alternativa
```
